## Feature
- Add your own shortcuts
- Easy configuration file
- Support different type of shortcuts

## Usage
- Enable the plugin and click `spaceship` in mini window to quick edit `~/.spaceship.json`
- Save & Restart `here` or Reload `spaceship`
